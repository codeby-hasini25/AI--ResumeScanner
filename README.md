# AI Resume Scanner

An advanced, enterprise-grade AI-powered Resume Parser and ATS (Applicant Tracking System) Evaluator. This application leverages modern natural language processing techniques to automate the tedious process of resume screening. By scanning resumes, extracting structured data, and comparing them against target job descriptions, this tool helps recruiters and hiring managers identify top-tier talent instantly.

---

## 🎯 About The Project

In today's job market, HR teams receive hundreds of applications for a single job opening. Manually filtering through these resumes is time-consuming and prone to human bias. 

The **AI Resume Scanner** solves this problem by acts as an automated screening assistant. It processes unstructured resume files (like PDFs and Word documents), categorizes information into logical entities, and runs a semantic comparison matrix against a given job description to find the perfect candidate match.

### Why Use This Tool?
* **Time Efficiency:** Reduces manual resume screening time by over 80%.
* **Objective Evaluation:** Standardizes the grading metric for all applicants.
* **Deep Insights:** Looks beyond basic keyword stuffing to understand the actual context of a candidate's experience.

---

## 🚀 Key Features

* **Automated Data Extraction:** Seamlessly parses contact information, social links (GitHub, LinkedIn), total years of experience, core skills, and educational background.
* **Intelligent Keyword & Semantic Matching:** Evaluates candidates based on synonyms, industry terms, and related technologies rather than just exact phrase matches.
* **Dynamic ATS Scoring Matrix:** Generates a comprehensive compatibility score (0-100%) showing how well the candidate fits the requirements.
* **Missing Keywords Identification:** Pinpoints exact gaps in skills or certifications between the resume and the job description.
* **Clean Structured Output:** Converts complex text documents into highly organized JSON data formats for easy integration with external databases.
